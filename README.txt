### TITLE ###
Supplementary materials including sample codes and data for the paper:
"Reconstructing the incidence rate and immune fraction of the population via a single snapshot survey: a case study of COVID-19 in Japan"
Submission to: PLOS Computational Biology

## Directories
- data/ – Contains census data
- r_code/ – R scripts for reading data, running inference on Stan
- stan_code/ – Stan scripts used in this study
- results/ - Contains results
